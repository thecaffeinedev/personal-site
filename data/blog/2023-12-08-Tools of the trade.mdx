**Software (Desktop Apps, CLI tools and Others)**  

- [fzf](https://github.com/junegunn/fzf "https://github.com/junegunn/fzf"): an awesome fuzzy finder written in `Go`!
- [ripgrep](https://github.com/BurntSushi/ripgrep "https://github.com/BurntSushi/ripgrep")**:** better replacement for `grep`. Also written in `Rust`.
- [bat](https://github.com/sharkdp/bat): A better version of `cat`
- [zoxide](https://github.com/ajeetdsouza/zoxide): A great tool for jumping between multiple directories.
- [exa](https://github.com/ogham/exa "https://github.com/ogham/exa") – a modern (?) replacement for `ls`. It’s been pretty good so far
- [Delta](https://github.com/dandavison/delta) (for showing pretty diffs in terminal)
- [pyenv](https://github.com/pyenv/pyenv "https://github.com/pyenv/pyenv"), [goenv](https://github.com/go-nv/goenv "https://github.com/go-nv/goenv"), [nvm](https://github.com/nvm-sh/nvm "https://github.com/nvm-sh/nvm"): for managing multiple versions of `python`, `go`, and `node` respectively.
- [rectangle](https://rectangleapp.com/ "https://rectangleapp.com/")**:** for organizing windows. Pretty sleek since I just need to press `⌘⌃←` to move current window to the left, etc.
- [BeeKeeper Studio:](https://www.beekeeperstudio.io/ "https://www.beekeeperstudio.io/") A good looking SQL GUI client for Postgres, SQLite and MySQL.
- [Amphetamine](https://apps.apple.com/in/app/amphetamine/id937984704?mt=12 "https://apps.apple.com/in/app/amphetamine/id937984704?mt=12"): I have been using it from past 3 years. It helps keep my Mac awake. 
- [MQTTX](https://mqttx.app/ "https://mqttx.app/"): GUI MQTT Client Toolbox. Works really great.
- [Postman](https://www.postman.com/downloads/ "https://www.postman.com/downloads/") App and [Thunder Client](https://marketplace.visualstudio.com/items?itemName=rangav.vscode-thunder-client "https://marketplace.visualstudio.com/items?itemName=rangav.vscode-thunder-client") (VSCode Extension)for API testing purpose. 
- [grpcurl](https://github.com/fullstorydev/grpcurl "https://github.com/fullstorydev/grpcurl"): Like cURL, but for gRPC
- [RedisInsight](https://redis.com/redis-enterprise/redis-insight/#insight-form "https://redis.com/redis-enterprise/redis-insight/#insight-form"):  GUI for redis client. Helps visually browse and interact with Redis data.
- [Docker](https://www.docker.com/ "https://www.docker.com/"): Container management.
- [HashiCorp Nomad:](https://www.nomadproject.io/ "https://www.nomadproject.io/")  Simple and flexible scheduler and orchestrator for managing containers. Kind of an alternative to Kubernetes.
- [ngrok](https://ngrok.com/) - A quick way to setup reverse shells and test complex environments locally.
